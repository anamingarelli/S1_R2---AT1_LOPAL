*Atividades em formato em formato de fluxograma:*

Exercício 1 → Desenvolvido para ler o nome do usuário e, a partir disso, exibir uma saudação personalizada.

Exercício 2 → O fluxograma solicitará ao usuário dois números, realizará a soma entre eles e apresentará o resultado ao final.

Exercício 3 → O fluxograma solicitará a idade do usuário, realizará a leitura do valor informado e verificará se a idade é menor, igual ou maior que 18 anos, classificando-o como maior ou menor de idade.

Exercício 4 → O usuário deverá inserir um número, e o sistema identificará se o valor informado é positivo, negativo ou igual a zero.

Exercício 5 → Ao informar duas notas, o sistema calculará a média final e exibirá o resultado ao usuário.
